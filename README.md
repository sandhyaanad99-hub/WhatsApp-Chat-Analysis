# WhatsApp-Chat-Analysis
A terminal-based Python data analytics tool that parses raw WhatsApp chat exports to extract and display behavioral insights through a beautifully formatted executive report.

## 1. Project Constraints & Guidelines Met
* **Language/Environment:** Built completely using Python 3 within a Google Colab / Jupyter Notebook environment.
* **No External Analytics Libraries:** All parsing, string manipulations, token counts, and mathematical calculations were achieved using pure Python standard libraries—strictly avoiding `pandas`, `numpy`, `regex`, or pre-built string/data mining packages.
* **Error & Edge-Case Handling:** Designed robust loops capable of cleanly identifying and appending multi-line WhatsApp message fragments back to their original senders without throwing index errors or structural breaks.
* **Output Standard:** Generates a highly aligned, beautifully padded text-based executive report matching the strict user-interface design specification layout.
